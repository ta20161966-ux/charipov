# Random Password Generator

## Автор
Иванов Иван

## Описание
Простое GUI-приложение для генерации случайных паролей с настройкой параметров и сохранением истории.

## Использование
1. Запустите `password_generator.py`.
2. Настройте параметры и сгенерируйте пароль.
3. История сохраняется в `history.json`.

## Примеры тестов
- Генерация пароля длиной 12 с цифрами и буквами.
- Генерация пароля только из спецсимволов.
- Проверка сохранения и загрузки истории.
